A small mono credit label — publication names, year stamps, status. Tracked-out uppercase mono.

```jsx
<Badge>L'Officiel</Badge>
<Badge variant="solid">Снобъ</Badge>
<Badge dot>Доступна для проектов</Badge>
```

Variants: `outline`, `solid`, `plain`. `dot` adds a tiny red marker spot (the only fill use of red).
