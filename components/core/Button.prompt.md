Editorial action button — uppercase Oswald, sharp corners. The brand's interactive accent lives in the `marker` variant (a red marker underline); never fill a button red.

```jsx
<Button variant="solid">Связаться</Button>
<Button variant="outline">Портфолио</Button>
<Button variant="marker" size="lg">Смотреть кейсы</Button>
<Button variant="ghost" size="sm">Скачать резюме</Button>
```

Variants: `solid` (black), `outline` (fills black on hover), `marker` (text + red underline), `ghost`. Sizes: `sm` / `md` / `lg`.
