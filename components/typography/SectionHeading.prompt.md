The signature heading: a giant uppercase Oswald word paired with a Playfair italic subhead — the brand's core typographic gesture.

```jsx
<SectionHeading index="01" title="PORTFOLIO" subtitle="Fashion & Creative" marker />
<SectionHeading title="EXPERIENCE" subtitle="Brand & Production" level={2} align="center" />
```

Use `marker` to draw the red underline under the title. `index` adds a tracked mono number above. Set `level` for correct document outline.
