A notebook page bound by a metal coil along one edge. Use for note cards, schedules, mood boards.

```jsx
<SpiralBinding edge="top" rings={14} tone="cream" style={{ maxWidth: 360 }}>
  <h3>Call sheet</h3>
  <p>09:00 — backstage / 11:00 — first look…</p>
</SpiralBinding>
```

`edge`: `top` (default), `bottom`, `left`, `right`. Pick `rings` to match the page width/height.
