A hang-tag swinging from a cord — for credits, on-set passes, name tags or price-tag labels. It swings from a metal grommet, so a small `rotate` reads naturally.

```jsx
<StringBadge tone="ink" rotate={-5}>
  <span className="label-micro" style={{ color: 'inherit' }}>BACKSTAGE</span>
  <strong style={{ fontFamily: 'var(--font-display)' }}>ACCESS&nbsp;ALL</strong>
</StringBadge>
```

Tones: `paper`, `cream`, `aged`, `ink` (inverted). Anchor the top of the cord wherever it should hang from.
